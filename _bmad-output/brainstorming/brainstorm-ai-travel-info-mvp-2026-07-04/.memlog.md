- (technique) started Job to Be Done
- (decision by user) Run brainstorming in chat using Facilitator Chosen + Creative Partner
- (insight by user) Core JTBD: plan the trip so users spend less time finding information
- (idea by user) MVP should cover route, hotels, sightseeing, food stops, road conditions, charging stations, parking, family-friendly stops, costs, safety, itinerary duration, and trip services like diving
- (idea by coach) Position MVP as time-saving trip planning from collected knowledge, not an AI travel encyclopedia
- (idea by coach) First promise: ask one question and receive a usable trip answer with sources and suggested next steps
- (idea by coach) Use a broad travel knowledge card model instead of separate polished feature UIs for each information type
- (idea by user) Initial sources: websites, possibly Google Maps later, Facebook group posts about XuyenViet, future user submissions, and service or partner sources like Agoda
- (technique) started One Feature Only
- (idea by coach) Prioritize a collector admin: paste URL, text, or image; AI extracts structured cards; human approves; cards become searchable by AI
- (decision by user) One public MVP feature should be AI Ask
- (idea by coach) AI Ask should answer in a travel-specific format: direct answer, suggested plan or options, relevant places and services, warnings or uncertainty, sources used, and follow-up questions
- (idea by user) Magic moment query: I want to plan a two-week trip from a given date with my two kids, destination could be HCMC, advise me
- (technique) started Role Playing
- (idea by coach) For broad under-specified planning questions, AI Ask should clarify, propose, and guide rather than only search
- (idea by coach) Magic moment answer should ask 2-3 missing questions, provide an initial two-week HCMC-oriented plan, and distinguish sourced info from general AI knowledge
- (idea by user) Helpful AI answer must consider kids, desired experience, and trips the user has already done
- (idea by coach) MVP can capture a tiny Trip Profile inside AI Ask: travelers, kids ages, desired experience, previous trips, dates, start point, possible destinations, budget, driving tolerance, and hotel style
- (insight by user) AI should prioritize user memory from conversation, then base knowledge, then search for updated information
- (insight by user) Collected internet and Facebook information may be incomplete or incorrect, so the MVP must handle uncertainty
- (technique) started Disney Method - Dreamer
- (idea by coach) Add a confidence layer to AI answers: based on user memory, from collected sources, needs checking, and confirm changing details before booking
- (idea by coach) Knowledge card minimum fields: title, type, location or route segment, summary, source, collected date, confidence, tags, freshness-sensitive flag
- (idea by user) Great parent-focused AI should understand parent goals and support children: shorter driving time, historical places, child discounts, and child-friendly experiences
- (technique) started Disney Method - Realist
- (idea by coach) Family-aware road-trip planning identity with kid comfort rules: driving limits, rest stops, learning plus fun plus rest, convenient hotel areas, no late check-ins, rainy-day backups, and honest boredom warnings
- (decision by user) Build AI Chat first with memory and personalization
- (idea by coach) Smallest build: Chat UI, memory extraction, admin knowledge cards, retrieval, answer template with confidence and sources, web search fallback later
- (technique) started Disney Method - Critic
- (insight by user) MVP fails if it does not provide useful information or practical tips
- (technique) started convergence with MoSCoW MVP scoping
- (decision by coach) Recommended MVP: AI Chat with memory and personalization, backed by curated knowledge cards and source/confidence-aware answers
- (insight by coach) Final chosen direction: AI Ask MVP for personalized Vietnam road-trip planning, with conversation memory, curated knowledge cards, retrieval, sources, confidence, and family-aware practical tips
